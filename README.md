# E-Commerce Delivery Performance Dashboard

This project analyzes delivery performance of e-commerce platforms using Excel.

## 📊 Key Features
- Revenue analysis by platform
- Delivery time evaluation
- Rating distribution
- Delay analysis

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## 📈 Key Insights
- Swiggy Instamart generates highest revenue
- Majority orders have higher ratings
- Delivery delays highlight improvement areas
